# Shadow Loop

Shadow Loop is a minimalist 2D puzzle-platformer with a time-loop mechanic.

## 🎮 Gameplay
- Use time loops and ghost shadows to solve puzzles
- 10 handcrafted levels
- Mobile-friendly controls
- Atmospheric visuals and ambient sound

## 📱 Platforms
- iOS (App Store Ready)
- Android (Coming Soon)
- WebGL (Optional via Unity Build Settings)

## 🛠 Built With
- Unity 2022.3 LTS
- C#
- Unity Cloud Build (for automated builds)

## 🚀 How to Run
1. Open with Unity 2022.3 LTS
2. Set target platform in Build Settings (iOS/Android/WebGL)
3. Press Play or Build and Run

- Unity Cloud Build
- App Store Connect
- GitHub CI/CD
